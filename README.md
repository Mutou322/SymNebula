# NebulaMind — 仿脑编程器
# 纯粹、确定性的数学公式推演与模型动态运行工具

仿脑编程器（NebulaMind）是一个纯粹、确定性的数学公式推演与模型动态运行工具。

它以人类大脑的神经元-突触网络为架构参考，以纯数学逻辑为唯一灵魂。用户通过创建"空神经元"、向其中输入自定义数学公式、并用"突触链接"将它们编织成星云图，即可让抽象的数学模型在计算机上演化、运行、被观察。

它不依赖任何人工智能、机器学习或智能辅助。它只是忠实执行数学逻辑的沙盒。

---

## 核心理念

- **神经元 = 公式节点**：每个节点完全由用户自定义公式赋予功能，无任何预设
- **突触 = 星云链接**：节点间的有向连接，传递数学量与约束
- **星云 = 整个逻辑网络**：去中心化、可无限延伸的抽象引用网
- **等式 = 数学平衡约束**：`=` 不是赋值，而是双向坍缩的约束关系
- **公式即源码**：数学公式本身就是驱动一切的源代码

## 技术架构

```
┌─────────────────────────┐
│   Python（纯UI层）       │  ← 只负责渲染、输入、观察窗
└───────────┬─────────────┘
            │
┌───────────▼─────────────┐
│   Rust（核心调度引擎）   │  ← 解析公式、管理星云图、调度计算
└───────────┬─────────────┘
            │
┌───────────▼─────────────┐
│   C/数学库（底层计算）   │  ← 纯数值运算，不参与任何决策
└─────────────────────────┘
```

| 层 | 职责 |
|----|------|
| **Python** | 纯皮肤与感官。渲染星云图、处理键鼠输入、驱动独立观察窗。不参与任何逻辑、解析、调度或计算。 |
| **Rust** | 唯一的大脑。管理全部图结构，解析数学公式为 AST，控制逻辑时钟，调度并行计算，处理状态机，管理工程文件。 |
| **C/数学库** | 纯粹的肌肉。执行矩阵运算、微积分、数值求解等。无任何决策能力。 |

## 核心特性

| 特性 | 说明 |
|------|------|
| **完全自定义** | 无内置公式库，从 1+1=2 到爱因斯坦场方程，全部由用户定义 |
| **双向约束求解** | 等式自动双向坍缩。已知 x 则求 y，反之亦然。多解时进入待定状态，等待用户附加限定条件 |
| **逻辑时钟驱动** | 支持反馈环路，通过离散时间步实现振荡、波动等动态演化 |
| **惰性求值** | 仅激活被观察或相关支路，极大降低资源消耗 |
| **符号作用域隔离** | 每个节点独立命名空间，λ 可同时表示波长和特征值，互不干扰 |
| **奇异点安全隔离** | 1/0、维度不匹配等异常自动标紫并切断输出，保护整体逻辑网 |
| **逻辑时钟与 UI 刷新分离** | 后台全速计算，界面独立采样，永不卡帧 |
| **跨平台便携** | 可装入 U 盘，无需安装，在不同系统上直接运行 |

| 格式 | XML + ZIP，后缀 `.brainstorm` |

---

## 节点状态体系

| 颜色 | 状态 | 含义 |
|------|------|------|
| 灰 | 静息/待编辑 | 未接入网络或尚未运行 |
| 黄 | 激活/待定 | 计算中或多解等待限定条件 |
| 绿 | 输出有效 | 计算成功，结果可传递给下游 |
| 紫 | 奇异态 | 数学异常（除零、维度不匹配、无解），已安全切断 |
